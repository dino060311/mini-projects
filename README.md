# Mini Projects

여기는 제가 코딩을 공부하면서 조금씩 만들어보고 있는 미니 프로젝트 모음입니다.

아직 혼자서 모든 코드를 짜기엔 막막할 때가 많습니다. 그래서 제미나이를 튜터처럼 활용하고 있습니다.  
단순히 코드를 복사해서 붙여넣는 것이 아니라, AI가 짜준 코드를 한 줄씩 뜯어보고, 모르는 부분은 다시 질문합니다.  
제 스스로 구조를 이해하고 넘어가는 방식으로 공부 중입니다.

---

## 프로젝트 목록

### 1. 전화 주문 인식 시스템 (Order-System)
* **설명**: 귀가 어두우신 할아버지를 위해 아이디어를 내서 만들어본 웹 화면입니다. 손님의 목소리를 인식해 텍스트로 변환한 뒤 화면에 크게 띄워줍니다.
* **사용 기술**: HTML, CSS, JavaScript (Web Speech API)
* **[프로젝트 자세히 보기](./Order-System)**

### 2. 스터디 트래커 (Study-Tracker)
* **설명**: Java와 데이터베이스를 연결해보는 연습을 위해 만든 공부 기록기입니다. DB를 이용해 회원가입, 로그인, 공부 시간 저장 및 누적 기록 조회 기능을 구현했습니다.
* **사용 기술**: Java, MySQL 8.0, JDBC
* **[프로젝트 자세히 보기](./Study-Tracker)**
